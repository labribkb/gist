Instructions

Zoomin by  drawing a rectangle  with your mouse over the area of interest. This will remove the overlap in that area with the specified tolerance.
Zoomout by right clicking on the view.
Clicking the Update button will recompute the current view according to the tolerance set with the slider.
If the whole dataset is display, the whole layout will be adjusted. If a portion of the dataset has been zoomed in, only the selected portion will be adjusted.
On large datasets, it is recommended to first adjust the whole layout with a moderate tolerance before zooming in on area of interest, with lower tolerances.
